# MR-khaled-abdelwahed
Khaled Academy Platform 🎓
A secure E-Learning platform for Arabic language education, featuring a modern UI and built with HTML, CSS, JS, and Firebase.

✨ Features:

👨‍🎓 Students: Watch video lectures, download PDFs, take timed quizzes, view grades, and receive live notifications. Accounts are highly secure and locked to a single device.
👨‍🏫 Admins: Full control to create courses, build exams, track student progress, reset device locks, and generate top-student leaderboards.
